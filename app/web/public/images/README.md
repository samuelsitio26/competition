# 📸 Image Assets Guide — MindHaven

Folder ini digunakan untuk menyimpan semua gambar yang digunakan di website MindHaven.

## Struktur Folder

```
images/
├── mental-health/     → Gambar tentang kesehatan mental (meditasi, mindfulness, wellness)
├── patients/          → Gambar pasien/mahasiswa (wajah tersenyum, aktivitas kampus)
├── socialization/     → Gambar sosialisasi & komunitas (grup diskusi, peer support)
├── support/           → Gambar dukungan & konseling (konselor, helpline, terapi)
├── team/              → Foto anggota tim pengembang
├── illustrations/     → Ilustrasi vektor/SVG untuk dekorasi halaman
├── icons/             → Custom icon atau favicon
└── backgrounds/       → Background pattern, texture, gradient images
```

## Panduan Penamaan File

- Gunakan **lowercase** dan **kebab-case**: `student-meditation.jpg`
- Sertakan ukuran jika ada variasi: `hero-banner-1920x1080.jpg`
- Format yang disarankan: `.webp` (terbaik), `.jpg`, `.png`, `.svg`

## Rekomendasi Gambar

### mental-health/
- Orang bermeditasi di alam terbuka
- Ilustrasi otak & mindfulness
- Journaling / menulis diary

### patients/
- Mahasiswa tersenyum di kampus
- Aktivitas belajar yang positif
- Ekspresi wajah beragam (representatif)

### socialization/
- Kelompok diskusi / sharing circle
- Teman saling mendukung
- Kegiatan komunitas kampus

### support/
- Konselor berbicara dengan mahasiswa
- Ilustrasi hotline / helpline
- Gambar tangan saling memegang (dukungan)

### illustrations/
- Vektor/SVG hero section
- Abstract shapes untuk background
- Ilustrasi fitur (mood tracker, assessment)

### backgrounds/
- Gradient mesh images
- Subtle pattern / texture
- Nature-themed backgrounds (calming)

## Sumber Gambar Gratis yang Direkomendasikan

- [Unsplash](https://unsplash.com) — Foto berkualitas tinggi gratis
- [Pexels](https://pexels.com) — Foto & video gratis
- [Undraw](https://undraw.co) — Ilustrasi SVG gratis (bisa custom warna)
- [Storyset](https://storyset.com) — Ilustrasi animasi gratis
- [Humaaans](https://humaaans.com) — Ilustrasi karakter manusia
- [Blush](https://blush.design) — Ilustrasi kustom gratis
